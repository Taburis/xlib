Hyperthesis Tests

A null hyperthesis H0H_0H0​ is a statement exclusive to the alternative hyperthese H1H_1H1​. A test is based on the samples X=(X1,…,Xn)\boldsymbol{X}=(X_1,\dots, X_n)X=(X1​,…,Xn​) from a random variable XXX defined on a probability space (Ω,R,P)(\Omega, \mathfrak{R},\mathbb{P})(Ω,R,P) and the support of XXX is S⊂Ω\mathcal{S}\subset\OmegaS⊂Ω. The test rule is

Reject H0(Accept H1),if X∈CRetain H0(Reject H1),if X∈Ω∖C.\begin{aligned} &\text{Reject } H_0 (\text{Accept } H_1),\quad \text{if } \boldsymbol{X}\in C\\ &\text{Retain } H_0 (\text{Reject } H_1),\quad \text{if } \boldsymbol{X}\in \Omega\setminus C. \end{aligned}​Reject H0​(Accept H1​),if X∈CRetain H0​(Reject H1​),if X∈Ω∖C.​

where C⊂SC\subset \mathcal{S}C⊂S is called critical region. The possible error is classified as

  • Type I error: H0H_0H0​ is rejected while it is true. It is quantified as the significance level or szie α\alphaα:
    α=max⁡H0PH0(X∈C),\alpha=\max_{H_0}\mathbb{P}_{H_0}(\boldsymbol{X}\in C),α=H0​max​PH0​​(X∈C),

  • Type II error: H0H_0H0​ is retained while it is false. It is quantified by 1−max⁡H1γC(H1)1-\max_{H_1}\gamma_C(H_1)1−maxH1​​γC​(H1​) where γC(H1)\gamma_C(H_1)γC​(H1​) is the power function of the critical region CCC for H1H_1H1​:
    γC(H1)=PH1(X∈C).\gamma_C(H_1)=\mathbb{P}_{H_1}(\boldsymbol{X}\in C).γC​(H1​)=PH1​​(X∈C).
