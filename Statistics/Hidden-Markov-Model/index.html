Hidden Markov Model

Definition#

A Hidden Markov Model (HMM) consisted by the following parts:

  1. Ground truth states, represented by a Markov process St=s1,…,sNS_t=s_1,\dots, s_NSt​=s1​,…,sN​.
  2. Observable events, represented by a Markov process Ot=o1,…,oNO_t=o_1,\dots,o_NOt​=o1​,…,oN​.
  3. Emission probability or observation probability P(Ot=ot∣S1=s1,…,St=st)=P(Ot=ot∣St=st)\mathbb{P}(O_t=o_t|S_1=s_1, \dots, S_t=s_t) = \mathbb{P}(O_t=o_t|S_t=s_t)P(Ot​=ot​∣S1​=s1​,…,St​=st​)=P(Ot​=ot​∣St​=st​). The emission probability can also be represented as a matrix EEE such that the element Ei,s=P(Ot=i∣St=s)E_{i,s}=\mathbb{P}(O_t = i|S_t = s)Ei,s​=P(Ot​=i∣St​=s).
  4. Tansition probability TTT which the (i,j)(i,j)(i,j) element Tij=P(Ot=j∣Ot−1=i)T_{ij}=\mathbb{P}(O_{t}=j|O_{t-1}=i)Tij​=P(Ot​=j∣Ot−1​=i) and then ∑iTij=1,∀j\sum_iT_{ij}= 1,\forall j∑i​Tij​=1,∀j.
  5. The initial probability distribution of truth states π\piπ that πi=P(S1=i)\pi_i = \mathbb{P}(S_1=i)πi​=P(S1​=i), ∑iπi=1\sum_i\pi_i=1∑i​πi​=1.

The HMM introduced here also assume that it is time-homogeneous which means that both transistion probability and emission probability is independent on the time subscripts.

The inference problem of HMM can be classified into three types:

  1. Given a observation sequence O={o1,…,oN}O=\lbrace o_1,\dots,o_N\rbraceO={o1​,…,oN​} and a HMM assumption Λ=(E,T)\Lambda=(E,T)Λ=(E,T), determine the likelihood P(O)\mathbb{P}(O)P(O).
  2. Given an observable sequence O={o1,…,oN}O=\lbrace o_1,\dots,o_N\rbraceO={o1​,…,oN​} and a assumption (E,T)(E,T)(E,T), discover the best hidden state sequence SSS.
  3. Given a set of observables and sequence of stats, learn the observation probability and transition probability (E,T)(E,T)(E,T).

Likelihood: Forward Algorithm#

For a HMM model with an assumption Λ=(E,T)\Lambda = (E,T)Λ=(E,T) and a sequnece of observations O={o1,…,on}O=\lbrace o_1, \dots, o_{n}\rbraceO={o1​,…,on​} is given, determine the likelihood LΛ(O)=P(O∣Λ)L_\Lambda(O)=\mathbb{P}(O|\Lambda)LΛ​(O)=P(O∣Λ) for obtaining the observation OOO with respect to the assumption Λ\LambdaΛ.

Based on the Markov assumption and the properties of conditional probability, it can be decomposed into

P(O)=∑kP(On=on∣Sn=k)P(Sn=k∣O1=o1,…,On−1=on−1),\mathbb{P}(O) = \sum_k\mathbb{P}(O_n = o_n|S_n=k)\mathbb{P}(S_n=k|O_1= o_1,\dots, O_{n-1} = o_{n-1}) ,P(O)=k∑​P(On​=on​∣Sn​=k)P(Sn​=k∣O1​=o1​,…,On−1​=on−1​),

where the last term on the right side can be expressed recursively like:

P(St=st∣O1=o1,…,Ot−1=ot−1)=∑st−1P(St−1=st−1∣O1=o1,…,Ot−2=on−2)P(st∣st−1)P(ot−1∣st)\mathbb{P}(S_t=s_t|O_1= o_1,\dots, O_{t-1} = o_{t-1}) = \sum_{s_{t-1}}\mathbb{P}(S_{t-1}=s_{t-1}|O_1= o_1,\dots, O_{t-2} = o_{n-2})\mathbb{P}(s_t|s_{t-1})\mathbb{P}(o_{t-1}|s_t)P(St​=st​∣O1​=o1​,…,Ot−1​=ot−1​)=st−1​∑​P(St−1​=st−1​∣O1​=o1​,…,Ot−2​=on−2​)P(st​∣st−1​)P(ot−1​∣st​)

since by definition we have:

P(St=st∣O1=o1,…,Ot=ot)=P(ot∣st)P(St=st∣O1=o1,…,Ot−1=ot−1).\mathbb{P}(S_t=s_t|O_1= o_1,\dots, O_{t} = o_{t})=\mathbb{P}(o_t|s_t)\mathbb{P}(S_t=s_t|O_1= o_1,\dots, O_{t-1} = o_{t-1}).P(St​=st​∣O1​=o1​,…,Ot​=ot​)=P(ot​∣st​)P(St​=st​∣O1​=o1​,…,Ot−1​=ot−1​).

This all bring us to a algorithm for solving this problem. Let's first define a parameter αt(st)\alpha_{t}(s_t)αt​(st​) as the probability of the ttt-th time step is sts_tst​ with the observations are fixed up to time step ttt:

αt(st)=P(St=st∣O1=o1,…,Ot=ot).\alpha_{t}(s_t) = \mathbb{P}(S_t=s_t|O_1= o_1,\dots, O_{t} = o_{t}).αt​(st​)=P(St​=st​∣O1​=o1​,…,Ot​=ot​).

We can summary the algorithm as follow:

  1. Initializing with α1(s1)=P(s1∣o1)=πs1Eo1,s1\alpha_{1}(s_1) = \mathbb{P}(s_1|o_1) = \pi_{s_1}E_{o_1, s_1}α1​(s1​)=P(s1​∣o1​)=πs1​​Eo1​,s1​​.
  2. Iteratively solving the αt(st)\alpha_{t}(s_t)αt​(st​) by the formula:
αt(st)=∑st−1αt−1(st−1)Tst−1,stEot,st,1≤t≤N.\alpha_{t}(s_t) = \sum_{s_{t-1}}\alpha_{t-1}(s_{t-1})T_{s_{t-1},s_t}E_{o_t, s_t}, \quad 1\le t\le N.αt​(st​)=st−1​∑​αt−1​(st−1​)Tst−1​,st​​Eot​,st​​,1≤t≤N.
  1. The likelihood P(O∣Λ)\mathbb{P}(O|\Lambda)P(O∣Λ) is then
P(O∣Λ)=∑snαn(sn).\mathbb{P}(O|\Lambda)=\sum_{s_n}\alpha_{n}(s_n).P(O∣Λ)=sn​∑​αn​(sn​).

Decoding: Viterbi Algorithm#

Given a sequence of observation O={o1,…,on}O=\lbrace o_1, \dots, o_{n}\rbraceO={o1​,…,on​} and a HMM assumption (E,T)(E,T)(E,T). Determine the most probable hidden stats.

The most probable hidden stats S={s1,…,sn}S=\lbrace s_1, \dots, s_{n}\rbraceS={s1​,…,sn​} are those maximizing the probability

vn(sn)=P(S1=s1,…,Sn=sn,O1=o1,…On=on).v_n(s_n) = \mathbb{P}(S_1=s_1,\dots, S_n=s_n,O_1=o_1,\dots O_n=o_n).vn​(sn​)=P(S1​=s1​,…,Sn​=sn​,O1​=o1​,…On​=on​).

Suppose vn−1(s)v_{n-1}(s)vn−1​(s) is known, then we have the relation

vn(sn)=max⁡sn−1vn−1(sn−1)Tsn−1,snEsn,on.v_n(s_n) = \max_{s_{n-1}}v_{n-1}(s_{n-1})T_{s_{n-1},s_n} E_{s_n,o_n}.vn​(sn​)=sn−1​max​vn−1​(sn−1​)Tsn−1​,sn​​Esn​,on​​.

A backtrace algorithm called Viterbi's Algorithm can be used to solve this problem by using the recurtion above. The pseudo code is

def Viterbi(o : observation_array):    T = transition_probability    E = emission_probability    p = initial_state_probability
    viterbi[M][N] # 2D array for total time step N and total candidate hidden states M 
    for s in hidden_states:        viterbi[s,1] = p[s]E(o[1], s)        backtrace[s,1] = 0    for t in range(2, N):        for s in hidden_states:            viterbi[s,t] = max( viterbi[s2 , t-1]*T[s, s2]*E(o[t], s) for s2 in hidden_states)            backtrace[s,t] = s2 # states that maximizing the element above    bestpathprob = max( viterbi[s, N] for s in hidden_states)    bestpathpointer = backtrace[s, N] # s is the one maximized the bestpathprob    bestpath = []    t = N    while not bestpathpointer == 0:        bestpath[t] = bestpathpointer        t-=1        bestpathpointer = backtrace[bestpathpointer, t] 
    return bestpath, bestpathprob

Learning: The Forward-Backward Algorithm#

Given sequences of observations and known the possible hidden state domain, learn the HMM parameters (E,T)(E,T)(E,T). The Baum-Welch algorithm is designed based on the idea of the large number theorem that the possible occurrence frequency is an unbiased estimator for these parameters.

T^x,y=number of states transfer x→ynumber of states transfer from x,E^x,k=number of observation of k, with hidden states xnumber of hidden states of x\begin{aligned} \hat T_{x,y}&=\frac{\text{number of states transfer } x\to y}{\text{number of states transfer from }x},\\ \hat E_{x,k}&=\frac{\text{number of observation of } k \text{, with hidden states } x}{\text{number of hidden states of }x} \end{aligned}T^x,y​E^x,k​​=number of states transfer from xnumber of states transfer x→y​,=number of hidden states of xnumber of observation of k, with hidden states x​​

Let's first define a parameter βt(st)\beta_t(s_t)βt​(st​) to be the probability of the observations after ttt is given as Ot+1=ot+1,…On=onO_{t+1}=o_{t+1},\dots O_n=o_nOt+1​=ot+1​,…On​=on​ provided the ttt-th hidden state is sts_tst​:

βt(st)=P(ot+1,…,on∣St=st).\beta_t(s_t) = \mathbb{P}(o_{t+1}, \dots, o_n|S_t=s_t).βt​(st​)=P(ot+1​,…,on​∣St​=st​).

This value can be calculated in a backward algorithm:

  1. Initialization with βn(x)=1\beta_n(x) = 1βn​(x)=1 for all candidate state xxx.
  2. Recusively calculate
βt(x)=∑yTx,yEy,ot+1βt+1(y),1≤t≤n.\beta_t(x) = \sum_{y}T_{x,y}E_{y,o_{t+1}}\beta_{t+1}(y), \quad 1\le t \le n.βt​(x)=y∑​Tx,y​Ey,ot+1​​βt+1​(y),1≤t≤n.
  1. The likelihood function with assumption Λ=(E,T)\Lambda = (E,T)Λ=(E,T) now can be expressed as
P(O∣Λ)=∑stαtstβt(st),1≤t≤n.\mathbb{P}(O|\Lambda) = \sum{s_t}\alpha_t{s_t}\beta_t(s_t),\quad 1\le t \le n.P(O∣Λ)=∑st​αt​st​βt​(st​),1≤t≤n.

Let the parameter ξt(x,y)\xi_t(x,y)ξt​(x,y) be the probability of the St=xS_t=xSt​=x and St+1=yS_{t+1}=ySt+1​=y provided the all observations are given under the assumption Λ\LambdaΛ:

ξt(x,y)=P(St=x,St+1=y∣O,Λ).\xi_t(x,y) =\mathbb{P}(S_t=x, S_{t+1}=y|O,\Lambda).ξt​(x,y)=P(St​=x,St+1​=y∣O,Λ).

With the conditional probability:

P(X,Y∣Z)=P(X∣Y,Z)P(Y∣Z),\mathbb{P}(X,Y|Z) = \mathbb{P}(X|Y,Z)\mathbb{P}(Y|Z),P(X,Y∣Z)=P(X∣Y,Z)P(Y∣Z),

we have

P(St=x,St+1=y∣O,Λ)=P(St=x,St+1=y,O,∣Λ)P(O∣Λ),\mathbb{P}(S_t=x, S_{t+1}=y|O,\Lambda) = \frac{\mathbb{P}(S_t=x, S_{t+1}=y,O,|\Lambda)}{\mathbb{P}(O|\Lambda)},P(St​=x,St+1​=y∣O,Λ)=P(O∣Λ)P(St​=x,St+1​=y,O,∣Λ)​,

which is

ξt(x,y)=αt(x)Tx,yEy,st+1βt+1(st+1)∑stαt(st)βt(st).\xi_t(x,y)=\frac{\alpha_t(x)T_{x,y}E_{y,s_{t+1}}\beta_{t+1}(s_{t+1})}{\sum_{s_t}\alpha_t(s_t)\beta_t(s_t)}.ξt​(x,y)=∑st​​αt​(st​)βt​(st​)αt​(x)Tx,y​Ey,st+1​​βt+1​(st+1​)​.

The estimator of T^x,y\hat T_{x,y}T^x,y​ is then

T^x,y=∑t=1n−1ξt(x,y)∑t=1n−1∑zξt(x,z).\hat T_{x,y} = \frac{\sum_{t=1}^{n-1}\xi_t(x,y)}{\sum_{t=1}^{n-1}\sum_z\xi_t(x,z)}.T^x,y​=∑t=1n−1​∑z​ξt​(x,z)∑t=1n−1​ξt​(x,y)​.

Similarly, define a parameter γt(st)\gamma_t(s_t)γt​(st​) as

γt(x)=P(St=x∣O,Λ)=αt(x)βt(x)P(O∣Λ)\gamma_t(x)=\mathbb{P}(S_t=x|O,\Lambda) = \frac{\alpha_t(x)\beta_t(x)}{\mathbb{P}(O|\Lambda)}γt​(x)=P(St​=x∣O,Λ)=P(O∣Λ)αt​(x)βt​(x)​

then we have the estimator

E^x,k=∑{Ot=k}γt(x)∑t=1nγt(x),\hat E_{x,k} = \frac{\sum_{\lbrace O_t=k \rbrace}\gamma_t(x)}{\sum_{t=1}^n\gamma_t(x)},E^x,k​=∑t=1n​γt​(x)∑{Ot​=k}​γt​(x)​,

where ∑{Ot=k}\sum_{\lbrace O_t=k \rbrace}∑{Ot​=k}​ means sum over all the observation samples with Ot=kO_t=kOt​=k. After we get the two estimators, we can express the algorithm as the following steps:

  1. Initializing the (E,T)(E,T)(E,T) with a hyperthesis values.
  2. Calculate the parameter α\alphaα from the forward algorithm and calculate β\betaβ from the backward algorithm.
  3. Calculate the γ,ξ\gamma, \xiγ,ξ from the new α,β\alpha,\betaα,β.
  4. Update the (E,T)(E,T)(E,T).
  5. Repeat the steps 2 to 4 until the parameters convergence.
