Baysian Estimation#

Suppose a random vector XXX stands for a sample drawn from a model Pθ\mathbb{P}_\thetaPθ​ with a parameter θ∈Θ\theta\in\Thetaθ∈Θ unknown, and e^=e(X)\hat e=e(X)e^=e(X) is a statistics used to estimate the value g(θ)g(\theta)g(θ). Then a loss function L(θ,X)\mathcal{L}(\theta,X)L(θ,X) represents the loss of the value estimated from e^\hat ee^ comparing with the truth g(θ)g(\theta)g(θ). It is natural to assume that L≥0\mathcal{L}\ge 0L≥0 and equal to 0 only if the estimation is the same as the truth. Furthermore, the risk function R(θ,e)R(\theta,e)R(θ,e) defined as

R(θ)=EL(θ,X),R(\theta)=\mathbb{E}\mathcal{L}(\theta,X),R(θ)=EL(θ,X),

which is the estimation of the loss over all the samples.

However, in the Baysian statistics, the possible value of the parameter θ\thetaθ is assumed to follow a prior distribution, say θ∼h\theta\sim hθ∼h. Then the model Pθ\mathbb{P}_\thetaPθ​ is consided as a conditional probability with the parameter is given, P(X∣θ)\mathbb{P}(X|\theta)P(X∣θ) and the estimator is the one minimizing the following expectation:

∫ΘR(θ)h(θ)dθ.\int_\Theta R(\theta)h(\theta)d\theta.∫Θ​R(θ)h(θ)dθ.

This type of estimator is called Bayes.

More formally, let (Ω,R,P)(\Omega, \mathcal{R},\mathbb{P})(Ω,R,P) be a probability space and define the following distributions:

  • X∼f(x∣θ)X\sim f(x|\theta)X∼f(x∣θ): the random variable following a PDF with a parameter θ\thetaθ undetermined.
  • Prior distribution θ∼h(θ)\theta\sim h(\theta)θ∼h(θ): an assumed distribution of the θ\thetaθ before observations.
  • Posterior distribution p(θ∣X)p(\theta|\boldsymbol{X})p(θ∣X): the realized distribution after samples X\boldsymbol{X}X were drawn.

The Bayes' theorem shows

P(X∈Ω∣θ=θ0)=P(X∈Ω,θ=θ0)P(θ=θ0).\mathbb{P}(X\in\Omega|\theta=\theta_0)= \frac{\mathbb{P}(X\in\Omega, \theta=\theta_0)}{\mathbb{P}(\theta=\theta_0)}.P(X∈Ω∣θ=θ0​)=P(θ=θ0​)P(X∈Ω,θ=θ0​)​.

These conceptions are connected by the Bayes' theorem as follows:

p(θ∣X)=L(X,θ)h(θ)P(X),L(X,θ)=∏i=1nf(Xi,θ),P(X)=∫ΘL(X,θ)dθp(\theta|\boldsymbol{X})= \frac{L(\boldsymbol{X},\theta)h(\theta)}{P(\boldsymbol{X})},\quad L(\boldsymbol{X},\theta)=\prod_{i=1}^nf(X_i,\theta),\quad P(\boldsymbol{X})=\int_\Theta L(\boldsymbol{X},\theta)d\thetap(θ∣X)=P(X)L(X,θ)h(θ)​,L(X,θ)=i=1∏n​f(Xi​,θ),P(X)=∫Θ​L(X,θ)dθ

where L(X,θ)h(θ)=P(X,θ)L(\boldsymbol{X},\theta)h(\theta) = \mathbb{P}(\boldsymbol{X}, \theta)L(X,θ)h(θ)=P(X,θ). The likelihood function L(X,θ)L(\boldsymbol{X},\theta)L(X,θ) represents the probability to obtain the sample X\boldsymbol{X}X for a fixed parameter θ\thetaθ. The P(X)P(\boldsymbol{X})P(X) is the marginal probability.

Remark:

Commonly, the parameter with prior distribution should not be viewed as the random in the frequentist sense. It reflects the information about the parameter before observations. With the observation made, the information is improved and reshape the prior distribution to the posterior distribtion. Similarly, the prior and posterior risk can be defined, respectively.

Based on the maximum likelihood assumption, the truth paramenter of θ\thetaθ should be the one maximize the probability p(θ∣X)p(\theta|\boldsymbol{X})p(θ∣X):

θ^=arg max⁡θ∈Θp(θ∣X)=arg max⁡θ∈ΘL(X,θ)h(θ),\hat \theta = \argmax_{\theta\in \Theta} p(\theta|\boldsymbol{X}) = \argmax_{\theta\in \Theta} L(\boldsymbol{X},\theta)h(\theta),θ^=θ∈Θargmax​p(θ∣X)=θ∈Θargmax​L(X,θ)h(θ),

the last equation follows from that the denominator L(X)L(\boldsymbol{X})L(X) has no θ\thetaθ-dependency. This estimation pick the peak of the p(θ∣X)p(\theta|\boldsymbol{X})p(θ∣X) as the prediction of θ\thetaθ. However, it is not the unique way to estimate parameters in Bayesian statistics. The idea of the Bayesian point estimation is to estimate the parameter θ^\hat\thetaθ^ is "close" to the truth θ\thetaθ measured by the loss function. This idea is illustrated by the following theorem:

Theorem:

Suppose e(X)e(X)e(X) is a statistics used for estimating the g(θ)g(\theta)g(θ) and

  1. EL(θ,X)<∞\mathbb{E}\mathcal{L}(\theta,X)<\inftyEL(θ,X)<∞ for some values of X\boldsymbol{X}X;
  2. The valude e^\hat ee^ defined as
    e^=arg min⁡θ∈Θ∫ΘL[θ,θ^(X)]dP(θ)=arg min⁡θ∈ΘE[L(θ,X=x)],\hat e = \argmin_{\theta\in\Theta}\int_{\Theta}\mathcal{L}[\theta, \hat \theta(\boldsymbol{X})]d\mathbb{P}(\theta) = \argmin_{\theta\in\Theta}\mathbb{E}[\mathcal{L}(\theta,\boldsymbol{X}=\boldsymbol{x})],e^=θ∈Θargmin​∫Θ​L[θ,θ^(X)]dP(θ)=θ∈Θargmin​E[L(θ,X=x)],
    exists almost everywhere for x\boldsymbol{x}x.

And L=∣θ−θ^∣\mathcal{L}=|\theta-\hat\theta|L=∣θ−θ^∣ would lead to the median of the p(θ∣X)p(\theta|\boldsymbol{X})p(θ∣X).

The estimation function can be interpreted as E(L∣X)\mathbb{E}(\mathcal{L}|\boldsymbol{X})E(L∣X). By definition, it can be expressed as

∫XE(L∣x)dP(x)=∫X∫ΘL(θ,θ^)p(θ∣x)P(x)dxdθ=∫X∫ΘL(θ,θ^)L(x∣θ)h(θ)dxdθ.\int_X\mathbb{E}(\mathcal{L}|\boldsymbol{x})d\mathbb{P}(\boldsymbol{x}) = \int_X\int_\Theta\mathcal{L}(\theta,\hat\theta)p(\theta|\boldsymbol{x}) P(\boldsymbol{x})d\boldsymbol{x}d\theta=\int_X\int_\Theta\mathcal{L}(\theta,\hat\theta)L(\boldsymbol{x}|\theta)h(\theta)d\boldsymbol{x}d\theta.∫X​E(L∣x)dP(x)=∫X​∫Θ​L(θ,θ^)p(θ∣x)P(x)dxdθ=∫X​∫Θ​L(θ,θ^)L(x∣θ)h(θ)dxdθ.

Then the risk function R(θ)R(\theta)R(θ) becomes

R(θ)=∫XL(θ,θ^)L(x∣θ)dx,R(\theta) = \int_X\mathcal{L}(\theta,\hat\theta) L(\boldsymbol{x}|\theta)d\boldsymbol{x},R(θ)=∫X​L(θ,θ^)L(x∣θ)dx,

the point estimator can be rewritten as

θ^=arg min⁡θE[R(θ)].\hat\theta = \argmin_{\theta}\mathbb{E}[R(\theta)].θ^=θargmin​E[R(θ)].
