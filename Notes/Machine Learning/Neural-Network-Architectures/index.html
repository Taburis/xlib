Max Pooling#

The function of the max pool layer is boosting the traing speed and preventing the over-fitting by reducing the outputs from selecting the maximum value among the inputs within a loca region as the outputs. For instance, for a size WÃ—HW\times HWÃ—H max pooling layer, it will slice the input into a many small region with shape WÃ—HW\times HWÃ—H, (padding or stride may apply). And only the maximum value of each slices will be the output from that region.
