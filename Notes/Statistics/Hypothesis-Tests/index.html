Hypothesis Tests

A null hypothesis H0H_0H0​ is a statement exclusive to the alternative hypotheses H1H_1H1​. The hyperthesis test is designed to ansewr the question that should the null hyperthesis be rejected? A test is based on the samples X=(X1,…,Xn)\boldsymbol{X}=(X_1,\dots, X_n)X=(X1​,…,Xn​) from a random variable XXX defined on a probability space (Ω,R,P)(\Omega, \mathfrak{R},\mathbb{P})(Ω,R,P) and the support of XXX is S⊂Ω\mathcal{S}\subset\OmegaS⊂Ω. The test rule is

Reject H0(Accept H1),if X∈CRetain H0(Reject H1),if X∈Ω∖C.\begin{aligned} &\text{Reject } H_0 (\text{Accept } H_1),\quad \text{if } \boldsymbol{X}\in C\\ &\text{Retain } H_0 (\text{Reject } H_1),\quad \text{if } \boldsymbol{X}\in \Omega\setminus C. \end{aligned}​Reject H0​(Accept H1​),if X∈CRetain H0​(Reject H1​),if X∈Ω∖C.​

where C⊂SC\subset \mathcal{S}C⊂S is called critical region. The possible error is classified as

  • Type I error: H0H_0H0​ is rejected while it is true (false positive). It is quantified as the significance level or szie α\alphaα:
    α=max⁡H0PH0(X∈C),\alpha=\max_{H_0}\mathbb{P}_{H_0}(\boldsymbol{X}\in C),α=H0​max​PH0​​(X∈C),
    The upper bound of the probability that X∈C\boldsymbol{X}\in CX∈C (rejecting H0H_0H0​) assuming H0H_0H0​ holds.
  • Type II error: H0H_0H0​ is retained while it is false (false negative). It is quantified as β=1−max⁡H1γC(H1)\beta=1-\max_{H_1}\gamma_C(H_1)β=1−maxH1​​γC​(H1​) where γC(H1)\gamma_C(H_1)γC​(H1​) is the power function of the critical region CCC for H1H_1H1​:
    γC(H1)=PH1(X∈C).\gamma_C(H_1)=\mathbb{P}_{H_1}(\boldsymbol{X}\in C).γC​(H1​)=PH1​​(X∈C).

A best critical region of size α\alphaα is a critical region CCC with size α\alphaα has the largest power amount all region with size α\alphaα.

Parametric Hypothesis Test#

A typical parametric hypothesis can be formulated as

H0:θ∈ω0H1:θ∈ω1,H_0: \theta\in \omega_0\quad H_1:\theta\in\omega_1,H0​:θ∈ω0​H1​:θ∈ω1​,

where ω0\omega_0ω0​ and ω1\omega_1ω1​ are subset of Ω\OmegaΩ and mutually disjoint.

For point wise parametric test:

H0:θ=θ0H1:θ=θ1,H_0: \theta=\theta_0\quad H_1:\theta=\theta_1,H0​:θ=θ0​H1​:θ=θ1​,

The random variable XXX used in the test derives a measure on the sample space SSS as

μX,θ(C)=∫CL(θ;x)dP(x),C∈S\mu_{X,\theta}(C) = \int_CL(\theta;\boldsymbol{x})d\mathbb{P}(\boldsymbol{x}), \quad C\in SμX,θ​(C)=∫C​L(θ;x)dP(x),C∈S

where L(θ;x)L(\theta;\boldsymbol{x})L(θ;x) is known as likelihood:

L(θ;x)=∏i=1nf(xi;θ),x=(x1,…,xn),L(\theta; \boldsymbol{x})=\prod_{i=1}^nf(x_i;\theta),\quad \boldsymbol{x}=(x_1,\dots,x_n),L(θ;x)=i=1∏n​f(xi​;θ),x=(x1​,…,xn​),

and f(x;θ)f(x;\theta)f(x;θ) is the PDF of XXX with the assumed parameter θ\thetaθ. With this notation, the size of a critical region is μX,θ0(C)\mu_{X,\theta_0}(C)μX,θ0​​(C) and the power is μX,θ1(C)\mu_{X,\theta_1}(C)μX,θ1​​(C). A test with the critical region CCC is unbiased if μX,θ1(C)≥μX,θ0(C)\mu_{X,\theta_1}(C)\ge\mu_{X,\theta_0}(C)μX,θ1​​(C)≥μX,θ0​​(C).

Using the linearity μX,θ1(C∪D)=μX,θ1(C)+μX,θ1(D)\mu_{X,\theta_1}(C\cup D)=\mu_{X,\theta_1}(C)+\mu_{X,\theta_1}(D)μX,θ1​​(C∪D)=μX,θ1​​(C)+μX,θ1​​(D) for any C∩D=∅C\cap D=\varnothingC∩D=∅ then the theorem follows.

Chi-Square Test#

Pearson's chi-squared test#

Given random variables of a multinomial distribution {Xk}\lbrace X_k\rbrace{Xk​}, which XiX_iXi​ stands for the occurrence of the iii-th outcome and kkk distinct outcomes in total, and assuming the test hypothesis is that

H0:pi=μi, ∀i=1,…,k.H1:otherwise.H_0: p_i=\mu_i,\: \forall i=1,\dots,k.\quad H_1: \text{otherwise}.H0​:pi​=μi​,∀i=1,…,k.H1​:otherwise.

The statistics for the hypothesis test is defined as

Q=∑i=1k(Xi−nμi)2nμi,∑iμi=1,Q∼χ2(k−1),Q=\sum_{i=1}^k\frac{(X_i-n\mu_i)^2}{n\mu_i},\quad\sum_i\mu_i=1,\quad Q\sim\chi^2(k-1),Q=i=1∑k​nμi​(Xi​−nμi​)2​,i∑​μi​=1,Q∼χ2(k−1),

and the critical region of size α\alphaα is determined according to the χ2(k−1)\chi^2(k-1)χ2(k−1) distribution.

Define Yi=(Xi−npi)/npiY_i=(X_i-np_i)/\sqrt{np_i}Yi​=(Xi​−npi​)/npi​​, then Yi∼N(0,1−pi)Y_i\sim N(0,1-p_i)Yi​∼N(0,1−pi​). Furthermore, for the covariance we have

Cov(Yi,Yj)=Cov(Xi/n,Xj/n)/(npipj)=−pipj\text{Cov}(Y_i,Y_j)=\text{Cov}(X_i/n,X_j/n)/(n\sqrt{p_ip_j})=-\sqrt{p_ip_j}Cov(Yi​,Yj​)=Cov(Xi​/n,Xj​/n)/(npi​pj​​)=−pi​pj​​

and hence Y∼Nk(0,Σ)\boldsymbol{Y}\sim N_k(0,\Sigma)Y∼Nk​(0,Σ) where Σii=1−pi\Sigma_{ii}=1-p_iΣii​=1−pi​ and Σij=−pipj\Sigma_{ij}=-\sqrt{p_ip_j}Σij​=−pi​pj​​. Noticing that if we introduce a vector v=(p1,…,pk)T\boldsymbol{v}=(\sqrt{p_1},\dots,\sqrt{p_k})^Tv=(p1​​,…,pk​​)T, then Σ=Ik−vvT\Sigma=I_k-\boldsymbol{v}\boldsymbol{v}^TΣ=Ik​−vvT. which proved the conclusion.

Fisher's Exact Test#

The fact that multinomial distribution approximates to normal distribution in large sample size makes the Chi-square test is not applicable to the case with limited sample size. The Fisher's exact test is introduced to small sample hypothesis test.

Given a nnn-size sample satisfying a multinomial distribution XiX_iXi​ according to features {Ak}\lbrace A_k\rbrace{Ak​} (XiX_iXi​ is the number of samples labeled as AiA_iAi​). Furthermore, these samples can also be labeled according a other feature set {Bh}\lbrace B_h\rbrace{Bh​} which satisfies a multinomial distribution denoted as YjY_jYj​. The hypothesis is

H0:Xi⊥YjH1:otherwise,H_0: X_i\perp Y_j \quad H_1:\text{otherwise},H0​:Xi​⊥Yj​H1​:otherwise,

and the random variable Pi,jP_{i,j}Pi,j​ stands for the counts for samples labeled as (i,j)(i,j)(i,j).

For the case that XXX and YYY are binomial, the distributions of Pi,jP_{i,j}Pi,j​ has only 4 possible combination and can be represented by a 2×22\times 22×2 table

A1A2Margin B
B1aba+b
B2cdc+d
Margin Aa+cb+dn=a+b+c+d

Fisher shows that Pi,jP_{i,j}Pi,j​ satisfies the hypergeometric distribution under the null hypothesis. The probability to have the table above is:

p=(a+ba)(c+dc)(na+c)=(a+bb)(c+dd)(nb+d)=(a+b)!(c+d)!(a+c)!(b+d)!a!b!c!d!n!.p=\frac{\binom{a+b}{a}\binom{c+d}{c}}{\binom{n}{a+c}}=\frac{\binom{a+b}{b}\binom{c+d}{d}}{\binom{n}{b+d}}=\frac{(a+b)!(c+d)!(a+c)!(b+d)!}{a!b!c!d!n!}.p=(a+cn​)(aa+b​)(cc+d​)​=(b+dn​)(ba+b​)(dc+d​)​=a!b!c!d!n!(a+b)!(c+d)!(a+c)!(b+d)!​.
danger

The probability ppp alone can't tell anything. To understand how rare of this sample, it is crucial to understand where this result located overall the hypergeometric distribution. The idea is that we need to calculate the probability to have a more extreme (imbalanced) result than the current one, which is sum over the probability of all extreme cases. It could be sum over the possible result imbalanced toward one-side or two-sides, depends on the problem we are facing. The cumulative probability of the extreme cases is the power or confidential to the null hypothesis.

The ttt-Test is a general term referring to all the hypothesis tests using a statistics following the ttt-distribution.

Z-Test#

Given a sample {Xn}\lbrace X_n\rbrace{Xn​} drawn from a distribution with mean μ\muμ and variance σ\sigmaσ. When the σ\sigmaσ is known, we define the statistics Z=(X‾−μ0)/σ/nZ=(\overline{X}-\mu_0)/\sqrt{\sigma/n}Z=(X−μ0​)/σ/n​, which has Z∼N(0,1)Z\sim N(0,1)Z∼N(0,1) assuming μ=μ0\mu=\mu_0μ=μ0​. And it can be used to test the hypothesis below (with the ppp-value followed):

  1. H0:μ≥μ0H_0: \mu\ge \mu_0H0​:μ≥μ0​, ppp-value: P(Z<z)=Φ(z)\mathbb{P}(Z<z)=\Phi(z)P(Z<z)=Φ(z) (left side tail)
  2. H0:μ≤μ0H_0: \mu\le \mu_0H0​:μ≤μ0​, ppp-value: P(Z>z)=1−Φ(z)=P(−z)\mathbb{P}(Z>z)=1-\Phi(z)=P(-z)P(Z>z)=1−Φ(z)=P(−z) (right side tail)
  3. H0:μ=μ0H_0: \mu=\mu_0H0​:μ=μ0​, ppp-value: P(∣Z∣<z)=2Φ(−∣z∣)\mathbb{P}(|Z|<z)=2\Phi(-|z|)P(∣Z∣<z)=2Φ(−∣z∣) (two side tails)

where zzz is the value of ZZZ obtained from the sample, and Φ(z)\Phi(z)Φ(z) is the cumulative function of standard normal distribution. The ppp-value stands for the extreme case of ZZZ assuming the null hypothesis is true. If μ≥μ0\mu\ge \mu_0μ≥μ0​, then extreme case that having a large ZZZ should not be small. It means tha a small ppp-value means that null hypothesis is unlikely to be true. The ppp-values picked for other two hypothesises based on the similar logic.

T-test vs Z-test

Usually the variance is unknown and needs to be estimated by sample variance S2/(n−1)S^2/(n-1)S2/(n−1). In this case, Z-test is an approximation to the t-test. Furthermore, the t-test is favored for the small sample size (30 or less) case.
