Reference

Given a set of samples {xi,yi}\lbrace x_i, y_i\rbrace{xi​,yi​}, the kernel method is used to analysis the pattern of the given samples.

Mathematical Foundation#

Reproducing Kernel Hilbert Space#

A Reproducing Kernel Hilbert Space (RKHS) H(X)\mathcal{H}(X)H(X) is a Hilbert space consists by functions defined on XXX such that Lx(f)L_x(f)Lx​(f) is continuous (or, equivalently, bouned) for any f∈Hf\in\mathcal{H}f∈H, where Lx:f→f(x)L_x:f\to f(x)Lx​:f→f(x) is a linear operator known as linear evalutaion functional. The continuous of LxL_xLx​ means that the small ∥f−g∥\Vert f-g\Vert∥f−g∥ implies that ∣f(x)−g(x)∣|f(x)-g(x)|∣f(x)−g(x)∣ is small for all x∈Xx\in Xx∈X.

Riesz Representation Theorem:

Let HHH be a Hilbert space whose inner product ⟨x,y⟩\langle x, y\rangle⟨x,y⟩ is linear for the second argument (yyy) and anti-linear for the first argument (xxx), then for every continuous linear functional φ∈H∗\varphi\in H^*φ∈H∗, there is a unique fφ∈Hf_\varphi\in Hfφ​∈H such that φ(f)=⟨fφ,f⟩\varphi(f)=\langle f_\varphi, f\rangleφ(f)=⟨fφ​,f⟩.

Based on this thoerem, the linear evalutaion functional can be expressed as

f(x)=Lx(f)=⟨Kx,f⟩,∀f∈H(X).f(x)=L_x(f)=\langle K_x,f\rangle, \quad \forall f\in \mathcal{H}(X).f(x)=Lx​(f)=⟨Kx​,f⟩,∀f∈H(X).

Moreover, Ly(Kx)=⟨Ky,Kx⟩=K(x,y)L_y(K_x)=\langle K_y,K_x \rangle=K(x,y)Ly​(Kx​)=⟨Ky​,Kx​⟩=K(x,y) and called K(x,y)K(x,y)K(x,y) as the reproducing kenerl and K:X×X→FK:X\times X\to \mathbb{F}K:X×X→F. It is symmetric and positive defined. And the point evaluation of any function f∈H(X)f\in\mathcal{H}(X)f∈H(X) can be expressed as

f(y)=∑iαiK(y,xi),f(y)=\sum_i\alpha_iK(y,x_i),f(y)=i∑​αi​K(y,xi​),

Moore-Aronszajn Theorem:

Suppose KKK is a postive defined symmetry kernel on XXX, then there is a unique Hilbert space of functions on XXX such that KKK is a reproducing kernel.

A gram matrix is defined as Kij=K(xi,xj)K_{ij}=K(x_i,x_j)Kij​=K(xi​,xj​). If the kernel is positive defined, then KijK_{ij}Kij​ is positive defined.

Support Vector Machine#

The Support Vector Machine (SVM) is an algorithm designed for solving the classification problem. Given a set of samples (xi,yi)(x_i,y_i)(xi​,yi​) where i=1,…,Ni=1,\dots,Ni=1,…,N and yi=±1y_i=\pm 1yi​=±1 labeling the sample into two classes, the SVM tries to find a hyper-surface to separate these classes.

Linear Support Vector Classification#

The linear support vector classifier (SVC) f(x)f(\boldsymbol{x})f(x) is a simplified SVM separating classes by a hyperplane determined by (n,b)(\boldsymbol{n}, b)(n,b), where n\boldsymbol{n}n is the vector normal to the hyperplane and bbb is a shift:

d(x,n,b)=0,d(\boldsymbol{x},\boldsymbol{n},b) =0,d(x,n,b)=0,

where n\boldsymbol{n}n is the vector normal to the hyperplane, b\boldsymbol{b}b is the offset, and

d(x,n,b)=n⋅x−b.d(\boldsymbol{x},\boldsymbol{n},b)=\boldsymbol{n}\cdot\boldsymbol{x}-b.d(x,n,b)=n⋅x−b.

Noticing that for a given sample x\boldsymbol{x}x, the distance from the itself to the hyperplane is given by d(x,n,b)/∥n∥d(\boldsymbol{x},\boldsymbol{n},b)/\Vert \boldsymbol{n}\Vertd(x,n,b)/∥n∥ and the sign of ddd indicates which side the point is located, the samples labeled by y=1y=1y=1 should be located on the side d(n,b)≥1d(\boldsymbol{n},b) \ge 1d(n,b)≥1, where 1 (it could be any constant) relates to margin of the hyperplane. Then the samples labeled by y=−1y=-1y=−1 should be on the other side defined as d(n,b)≤1d(\boldsymbol{n},b)\le 1d(n,b)≤1. The margin width is 2/∥n∥2/\Vert\boldsymbol{n}\Vert2/∥n∥.

The linear SVM f(x∣n,b)f(\boldsymbol{x}|\boldsymbol{n},b)f(x∣n,b) is defined as

f(x)=Θ(d(x,n,b)−1)−Θ(d(x,n,b)+1),f(\boldsymbol{x}) = \Theta(d(\boldsymbol{x},\boldsymbol{n},b)-1)-\Theta(d(\boldsymbol{x},\boldsymbol{n},b)+1),f(x)=Θ(d(x,n,b)−1)−Θ(d(x,n,b)+1),

where Θ(x)\Theta(x)Θ(x) is a step function vanish for x<0x<0x<0 and Θ(x)=1\Theta(x)=1Θ(x)=1 for x≥0x\ge 0x≥0.

Given a sample (x,y)(\boldsymbol{x},y)(x,y), a correct classification will have yf(x)≥1yf(\boldsymbol{x})\ge 1yf(x)≥1 while the mis-classification leads to yf(x)≤−1yf(\boldsymbol{x})\le -1yf(x)≤−1. The loss function of the linear SVM can be constructed as

L(n,b)=∑i=1N−yif(x)+λ∥n∥2/4,\mathcal{L}(\boldsymbol{n},b)=\sum_{i=1}^N-y_if(\boldsymbol{x})+\lambda\Vert \boldsymbol{n}\Vert^2/4,L(n,b)=i=1∑N​−yi​f(x)+λ∥n∥2/4,

where λ\lambdaλ is an arbitrary parameter and the second term is the square of the inverse of the margin width 2/∥n∥2/\Vert\boldsymbol{n}\Vert2/∥n∥. The fit minimizing this loss by gradient descend will ping down the parameters of the hyperplane. The loss function L\mathcal{L}L can be viewed as maximizing the margin width (second term) with a constraint on the hyperplane (first term) by using a Lagrange multiplier method. On the other hand, it could also viewed as a cross-entropy with a L2 regularization.

Linear Support Vector Regression#

Unlike the linear SVC trying to separate the samples outside the hyperplane margin, the linear support vector regression (SVR) are trying to find a hyperplane with a boundary to contain as much samples as possible. In this case, the target value yyy is represented by the offset of the hyperplane and

f(x)=⟨n,x⟩+b,f(\boldsymbol{x})=\langle \boldsymbol{n},\boldsymbol{x}\rangle+b,f(x)=⟨n,x⟩+b,

which is the same form as a linear regression. Furthermore, we are expecting

∣f(xi)−yi∣≤ϵ+ξi,|f(\boldsymbol{x}_i)-y_i|\le \epsilon+\xi_i,∣f(xi​)−yi​∣≤ϵ+ξi​,

where 2ϵ/∥n∥2\epsilon/\Vert\boldsymbol{n}\Vert2ϵ/∥n∥ is the width of the boundary and ξi\xi_iξi​ is a tolerance for iii-th samples. The hyperplane is determined by minimizing the loss

L=∑iφ(yi−f(xi))+λ∥n∥2,\mathcal{L} = \sum_i\varphi(y_i-f(\boldsymbol{x}_i))+\lambda\Vert \boldsymbol{n}\Vert^2,L=i∑​φ(yi​−f(xi​))+λ∥n∥2,

where the φ\varphiφ is a loss function customizable. In the case of φ(x)=x2\varphi(x)=x^2φ(x)=x2, it leads to a least square estimation with a L2 regularization. It could also be φ(x)=max(0,∣x∣−ϵ)\varphi(x)=\text{max}(0, |x|-\epsilon)φ(x)=max(0,∣x∣−ϵ) that penalty only comes from the points outside the boundary.
