Lebesgue Integral

Definitions#

The logic of the definition of the Lebesgue integral is that the lebesgue integral can be defined clearly on simple function, and extend it to general function by approximating the general function by a sequence of simple functions.

Definition on Simple Functions#

Given a measure μ\muμ defined on a ring Sμ\mathfrak{S}_{\mu}Sμ​ over a set XXX, a simple function is a μ\muμ-measurable function f:X→Rf:X\to \mathbb{R}f:X→R that has only countable distinct values. The Lebesgue integral of a simple function fff over a set AAA is defined as the summation following if it was convergent absolutely:

∫Af(x)dμ=∑yiyiμ(Xyi),Xyi={x∈A∣f(x)=yi}.\int_Af(x)d\mu = \sum_{y_i}y_i\mu(X_{y_i}), \quad X_{y_i}=\lbrace x\in A| f(x) = y_i\rbrace.∫A​f(x)dμ=yi​∑​yi​μ(Xyi​​),Xyi​​={x∈A∣f(x)=yi​}.

A simple function is called μ\muμ-integrable if its integral exists.

The Lebesgue integral of simple functions have the following properties:

  1. Suppose A=∪iBiA=\cup_i B_iA=∪i​Bi​ where Bi∩Bj=∅B_i\cap B_j=\varnothingBi​∩Bj​=∅ and fff has only one value, say ckc_kck​ over BkB_kBk​, then the sufficient and necessary condition for the integral of fff to exist is that the following summation converent absolutely

General Definition#

The Lebesgue integral for a μ\muμ-measurable function fff is defined as the following limit if it exists:

∫Af(x)dμ=lim⁡n→∞∫Afn(x)dμ,\int_Af(x)d\mu = \lim_{n\to\infty}\int_Af_n(x)d\mu,∫A​f(x)dμ=n→∞lim​∫A​fn​(x)dμ,

where {fn}\lbrace f_n\rbrace{fn​} is a sequence of simple μ\muμ-measurable functions convergent to f(x)f(x)f(x) uniformly.

Remark: This definition is valid since the following conditions hold:

  1. The limit exists for any fnf_nfn​ uniformly convergent to fff. Suppose fn,fmf_n,f_mfn​,fm​ are functions uniqformly convergenet to fff, then

  2. The integral is independent on the choice of fnf_nfn​. Suppose fnf_nfn​ and gmg_mgm​ are two sequneces of simple functions convergent to fff uniformly, then
    ∣∫Afn(x)dμ−∫Agm(x)dμ∣≤μ(A)sup{∣fn(x)−f(x)∣:x∈A}+μ(A)sup{∣gm(x)−f(x)∣:x∈A}.\left |\int_A f_n(x)d\mu -\int_A g_m(x)d\mu\right| \le \mu(A)\text{sup}\lbrace|f_n(x)-f(x)|: x\in A\rbrace+\mu(A)\text{sup}\lbrace|g_m(x)-f(x)|: x\in A\rbrace.∣∣∣∣∣​∫A​fn​(x)dμ−∫A​gm​(x)dμ∣∣∣∣∣​≤μ(A)sup{∣fn​(x)−f(x)∣:x∈A}+μ(A)sup{∣gm​(x)−f(x)∣:x∈A}.
  3. It is obviously that this definition is the same as the Lebesgue integral definition for simple functions.

    which means that f(x)f(x)f(x) is integrable.
